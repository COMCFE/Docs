# 商品管理

# 列表[重写]

方式

GET

路径

/admin/goods

参数

名称 类型 默认 必须 说明
name string 无 否 名称
category_id int 无 否 分类名称
limit int 15 否 每页条数
status int 无 否 上下架 可选参数: 1-上架 2-下架
delete int 无 否 回收站 可选参数: 0-正常 2-删除
usable string 无 否 传1时表示获取未绑定商户的商品
