# 订单

## 创建订单

**方式**

`POST`

**参数**

|    名称    |  类型  | 必须 |                     说明                      |
| :--------: | :----: | :--: | :-------------------------------------------: |
|  goods_id  |  int   |  是  |                    商品 ID                    |
|    num     |  int   |  是  |                   购买数量                    |
| deduct_num |  int   |  否  | CL 抵扣数量.不进行抵扣或者抵扣为 0 不传此参数 |
|   pay_password   | string |  是  |                   支付密码                    |

**路径**

`/api/orders/usdt`

**响应**

`Status code 200`

```json
{
  "msg": "ok",
  "code": 0,
  "data": {
    "order_no": "32523393" /*附言码/订单号*/
  }
}
```
