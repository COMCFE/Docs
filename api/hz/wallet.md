# 获取充值记录

## 获取充值记录

**方式**

`GET`

**参数**

|  名称  |  类型  | 必须 |               说明               |
| :----: | :----: | :--: | :------------------------------: |
| currency | string | 必须 | usdt/eoc |

**路径**

`/api/recharge/records`

**响应**

`Status code 200`

```json
{
  "msg": "ok",
  "code": 0,
  "data": null
}
```
