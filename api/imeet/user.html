# 用户登录

方式

POST

路径

/api/auth/login

参数

名称 类型 必须 说明
phone string 是 手机号
password string 否 密码 如type为pass必须
code string 否 验证码 如type为code必须
ticket string 否 防水墙票据 如type为pass必须
randstr string 否 防水墙随机串 如type为pass必须
type string 是 登录方式: pass,code

响应

Status code 200

{
    "msg": "ok",
    "code": 0,
    "data": {
        "token": "xxxxxxxxxx..."
    }
}
