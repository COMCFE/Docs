# 矿石

## 挖矿列表

`/api/v1/ores`

**请求方式**

`GET`

**请求参数**

| 名称  | 类型 | 默认 | 必须 | 说明 |
| :---: | :--: | :--: | :--: | :--: |
| limit | int  |  20  |  否  | 条数 |

**返回体**

```json
{
  "msg": "",
  "code": 0,
  "data": [
    {
      "id": 34 /*id*/,
      "num": "0.25520000" /*数量*/
    }
  ]
}
```

## 同步记录

`/api/v1/ores`

**请求方式**

`POST`

**请求参数**

`无`

**返回体**

```json
{
  "msg": "拉取成功",
  "code": 0,
  "data": null
}
```

## 领取矿石

`/api/v1/ores`

**请求方法**

`PUT`

**请求参数**

| 名称 | 类型  | 默认 | 必须 | 说明 |
| :--: | :---: | :--: | :--: | :--: |
| ores | array |  无  |  是  |  ID  |

**返回体**

```json
{
  "msg": "成功",
  "code": 0,
  "data": null
}
```
